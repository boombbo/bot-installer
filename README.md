# GitHub Repository Manager

GitHub Repository Manager is a lightweight Electron app designed for easy management and cloning of GitHub repositories. It offers a user-friendly interface for adding, removing, and installing repositories directly from GitHub.

## Features

- **Add Repositories**: Easily add GitHub repository URLs to your list.
- **Remove Repositories**: Remove repositories from your list with a simple click.
- **Clone Repositories**: Clone selected repositories to a local directory of your choice.

## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed: 
- [Node.js](https://nodejs.org/)
- [Git](https://git-scm.com/) 

### Installation

To get a local copy up and running, follow these simple steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/boombbo/bot-installer.git  
   ```

2. Navigate to the project directory:
   ```bash
   cd bot-installer
   ```

3. Install NPM packages:
   ```bash
   npm install
   ```

4. Start the application: 
   ```bash
   npm start
   ```
or click 

## Usage

After starting the application, you will be presented with a simple UI.

- To add a repository, enter the repository's URL in the input field and click "添加仓库".
- To remove repositories, select the repositories using the checkboxes and click "删除所选仓库".
- To clone repositories, select the repositories and click "安装所选仓库", then choose the directory where you want to clone them.
